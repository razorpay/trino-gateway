# AWS ALB Metrics

Read this file when the service is fronted by an AWS Application Load Balancer and ALB visibility is part of the standard coverage.

## Metric Families

| Metric family | Metric | Description |
|---|---|---|
| Request count | `aws_alb_request_count` | Request volume through the ALB |
| Target response time | `aws_alb_target_response_time` | Response time seen at the target side |
| ELB HTTP error codes | `aws_alb_httpcode_elb` | Errors generated by the load balancer itself |
| Target HTTP error codes | `aws_alb_httpcode_target` | Errors returned by backend targets |
| Healthy host count | `aws_alb_healthy_host_count` | Number of healthy registered targets |

## Baseline Expectation

When an ALB fronts the service, request volume, target latency, load-balancer errors, target errors, and healthy-target count should be visible in the standard infra view.
